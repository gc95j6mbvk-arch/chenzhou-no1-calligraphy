# 湖南省郴州市第一中学书法社官网

这是一个可上线的 Node.js 全栈官网项目，包含：

- 前台官网：首页、社团介绍、公告、书法作品展、活动记录、成员风采、招新报名、作品上传、联系我们。
- 后台管理：基础信息、公告、活动、成员、作品内容管理；报名记录查看；作品投稿审核。
- 文件存储：报名、内容和投稿先保存在 `data/*.json`，上传图片保存在 `public/uploads/`。

## 本地运行

```bash
node server.js
```

默认访问地址：

- 前台：http://localhost:3000
- 后台：http://localhost:3000/admin.html

默认后台密码：

```text
calligraphy2026
```

## 上线配置

上线时建议设置环境变量：

```bash
PORT=3000
ADMIN_PASSWORD=换成强密码
ADMIN_SECRET=换成一串随机密钥
```

支持部署到能运行 Node.js 的平台，例如学校服务器、Render、Railway、Fly.io 或普通云服务器。

详细上线和 `.com` 域名绑定步骤见 `DEPLOY.md`。

## 内容替换

真实资料准备好后，可以在后台直接替换：

- 社团成立时间
- 指导老师
- 社长/干部
- QQ 群二维码
- 校园、活动、作品图片
- 公告和活动文章

作品上传后默认进入“投稿审核”，管理员点击“通过并展示”后才会出现在前台作品展。
